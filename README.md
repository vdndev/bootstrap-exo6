# bootstrap-exo6
Exercice d'application de Bootstrap

## Usage

Version minifié dans /dist

## Development

```bash
git clone https://github.com/vdndev/bootstrap-exo6.git
```

```bash
npm install
```

```bash
npm run dev
```

Puis ouvrez la [page](http://localhost:5174/)

## Build

```bash
npm run build
```

La version minifiée sera dans src/dist
